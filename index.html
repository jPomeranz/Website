My Projects

Gimgur

Gimgur is a application developed with the help of a few friends as an entry into the 2015 Hack.UVA Hackathon. The application draws inspiration from Gyazo and allows for instantaneous, one-click uploading of photos to the popular image sharing website Imgur. The application was coded in C++ and uses the original Imgur API key system, since implementing OAuth2 support was outside of the scope of the project. The application was a Hack.UVA 2015 Finalist and winner of the Best Web App award. More information is available here.

Automatic Bike Shifter

As one of my Rodman design projects, I worked with a team of students to design an automatic bike shifter. Using an Arduino Uno, a Hall Effect sensor, and a N52 magnet on a wheel, the optimum gear necessary to maintain the current pedaling speed is calculated in real-time. Due to time constraints for the project, the results are displayed to a set of LEDs instead of servos.

TA Forum

The TA Forum was designed as an alternative to The Course Forum with a focus on TA's and reviews, as opposed to professors. The project focused mainly on the relational database design, but also included a fair amount of front-end work in PHP, jQuery, and Bootstrap.
